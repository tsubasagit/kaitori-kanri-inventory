# QRコード / バーコード運用ガイド

## 概要

本システムでは、古着の買取時に商品ごとにQRコード（またはバーコード）を貼り付け、
販売時にそのコードをカメラで読み取ることで商品を特定し、カートに追加します。

```
【買取時】                    【販売時】
商品を査定                    QRコードをスキャン
  ↓                            ↓
システムに登録                商品が自動でカートに追加
  ↓                            ↓
バーコード発行                 会計・決済
(HKM-20260315-0001)
  ↓
QRコード/バーコードを
ラベルに印刷して商品に貼付
```

---

## QRコードの作成方法

商品登録時にシステムが自動で `HKM-YYYYMMDD-XXXX` 形式のバーコード番号を発行します。
この番号をQRコードにして商品タグに貼り付けます。

### 方法1: 無料Webサイトで作成（最も手軽）

以下のサイトにアクセスし、バーコード番号を入力するだけでQRコードが生成されます。

| サイト名 | 特徴 |
|---|---|
| QRのススメ (qr.quel.jp) | 日本語、登録不要、無料 |
| くるくる Manager (m.qrqrq.com) | まとめて作成可能 |

**手順:**
1. サイトにアクセス
2. 「テキスト」を選択
3. バーコード番号（例: `HKM-20260315-0001`）を入力
4. 「作成」ボタンを押す
5. 生成されたQRコード画像をダウンロード
6. ラベルシールに印刷して商品に貼付

### 方法2: Excelで一括作成

大量の商品を扱う場合に便利です。

**手順:**
1. Excelを開く
2. A列にバーコード番号を一覧で入力
3. B列に以下の数式を入力（Google Charts APIを利用）:
   ```
   ="https://chart.googleapis.com/chart?cht=qr&chs=150x150&chl="&A1
   ```
4. B列のURLをブラウザで開くとQRコード画像が表示される
5. まとめて印刷

### 方法3: スマートフォンアプリで作成

| アプリ名 | OS | 特徴 |
|---|---|---|
| QRコードリーダー & 作成 | iOS / Android | 読み取り・作成両対応 |
| QR Code Generator | iOS / Android | シンプルで使いやすい |

**手順:**
1. アプリをインストール
2. 「作成」タブを選択
3. 「テキスト」を選択し、バーコード番号を入力
4. QRコードが生成される
5. 画像を保存 → ラベルシールに印刷

### 方法4: ラベルプリンターで直接印刷

Brother QL-800 などのラベルプリンターをお持ちの場合:

1. 付属ソフト（P-touch Editor 等）を起動
2. 「バーコード」挿入 → QRコード形式を選択
3. バーコード番号を入力
4. ラベルに直接印刷

## ラベルの貼り方

### 推奨ラベル
- **サイズ**: 30mm x 30mm 以上（小さすぎるとカメラで読み取りにくい）
- **素材**: 耐水性のあるラベルシール推奨
- **用紙**: A4ラベルシール（100均でも購入可能）

### 貼付位置
| 商品カテゴリ | 推奨貼付位置 |
|---|---|
| トップス | 内側の洗濯タグ付近 / 値札タグ |
| ボトムス | ウエスト内側タグ付近 / 値札タグ |
| アウター | 内ポケット付近 / 値札タグ |
| シューズ | 靴底の裏 / 箱がある場合は箱 |
| バッグ | 内ポケット / 値札タグ |
| アクセサリー | 台紙 / 値札タグ |

### ラベルに記載する情報（推奨）
```
┌─────────────────────┐
│  [QRコード]  ¥4,980  │
│                      │
│  HKM-20260315-0001   │
│  Levi's 501 / M      │
└─────────────────────┘
```

QRコードに加えて、以下を印字しておくと便利です:
- 販売価格
- バーコード番号
- 商品名（簡略）/ サイズ

## 販売時のスキャン方法

1. 販売（レジ）画面を開く
2. **「QR / バーコードで追加」** ボタンをタップ
3. カメラが起動する
4. 商品に貼られたQRコードを枠内に合わせる
5. 自動で読み取り → 商品がカートに追加される
6. 複数商品がある場合は再度スキャンボタンをタップ
7. 全商品を追加したら「会計へ進む」

### スキャンがうまくいかない場合
- QRコードが汚れていないか確認
- カメラとQRコードの距離を10〜20cmに調整
- 照明が十分か確認（暗いと読み取りにくい）
- 手動検索に切り替えてバーコード番号を入力

## よくある質問

**Q: QRコードとバーコード、どちらを使えばよいですか？**
A: QRコードを推奨します。バーコード（CODE128等）も読み取れますが、QRコードの方が小さく印刷しても読み取り精度が高いです。

**Q: QRコードの中身は何を入れればよいですか？**
A: システムが発行するバーコード番号（例: `HKM-20260315-0001`）をそのまま入力してください。URLにする必要はありません。

**Q: 同じQRコードを2枚作ってしまいました**
A: 問題ありません。同じ番号は1つの商品にのみ紐づくため、どちらをスキャンしても同じ商品が表示されます。

**Q: QRコードが剥がれてしまいました**
A: 在庫管理画面からバーコード番号を確認し、新しいQRコードを作成して貼り直してください。

**Q: 対応ブラウザは？**
A: QRコード読み取り機能は **Google Chrome** または **Microsoft Edge** の最新版で動作します。Safari（iOS）は現在非対応です。

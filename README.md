# 古着在庫管理システム

古着買取・販売店向けの店舗管理Webアプリケーションです。
買取時にQRコードを発行・貼付し、販売時にカメラでスキャンして商品を特定する運用フローに対応しています。

## 運用フロー

```
【買取】                         【販売】
お客様から古着を査定               QRコードをカメラでスキャン
  ↓                                ↓
カメラで商品写真を撮影             商品が自動でカートに追加
  ↓                                ↓
システムに商品情報を登録            決済方法を選択して会計
  ↓
バーコード番号を自動発行
(HKM-20260315-0001)
  ↓
QRコードを作成して商品に貼付
```

## 主な機能

| 機能 | 説明 |
|---|---|
| **買取（仕入れ）** | 顧客選択 → カメラで商品撮影 → QRコード読み取り → 査定入力 → 買取確定 |
| **販売（レジ）** | QRコードスキャンで商品をカートに追加 → 決済（現金/クレカ/電子マネー/QR決済） |
| **在庫管理** | 商品一覧・検索・フィルタ・ステータス管理（在庫あり/販売済み/返品/廃棄） |
| **顧客管理** | 顧客情報の登録・検索・本人確認記録（古物営業法対応） |
| **取引履歴** | 全取引の一覧・検索・レシート表示 |
| **ダッシュボード** | 本日の売上/買取サマリ・直近取引・クイックアクション |

## QRコード / バーコード運用ガイド

### 仕組み

本システムでは、買取時に商品ごとにQRコードを貼り付け、販売時にそのコードをカメラで読み取ることで商品を特定し、カートに追加します。

- **買取時**: システムが `HKM-YYYYMMDD-XXXX` 形式のバーコード番号を自動発行
- **販売時**: 商品に貼られたQRコードをカメラでスキャンするだけで商品特定

### QRコードの作成方法

商品登録時にシステムが自動でバーコード番号を発行します。
この番号をQRコードにして商品タグに貼り付けてください。

#### 方法1: 無料Webサイトで作成（最も手軽）

以下のサイトにアクセスし、バーコード番号を入力するだけでQRコードが生成されます。

| サイト名 | 特徴 |
|---|---|
| QRのススメ (qr.quel.jp) | 日本語、登録不要、無料 |
| くるくる Manager (m.qrqrq.com) | まとめて作成可能 |

**手順:**
1. サイトにアクセス
2. 「テキスト」を選択
3. バーコード番号（例: `HKM-20260315-0001`）を入力
4. 「作成」ボタンを押す
5. 生成されたQRコード画像をダウンロード
6. ラベルシールに印刷して商品に貼付

#### 方法2: Excelで一括作成

大量の商品を扱う場合に便利です。

1. Excelを開く
2. A列にバーコード番号を一覧で入力
3. B列に以下の数式を入力:
   ```
   ="https://chart.googleapis.com/chart?cht=qr&chs=150x150&chl="&A1
   ```
4. B列のURLをブラウザで開くとQRコード画像が表示される
5. まとめて印刷

#### 方法3: スマートフォンアプリで作成

| アプリ名 | OS | 特徴 |
|---|---|---|
| QRコードリーダー & 作成 | iOS / Android | 読み取り・作成両対応 |
| QR Code Generator | iOS / Android | シンプルで使いやすい |

1. アプリをインストール
2. 「作成」→「テキスト」を選択
3. バーコード番号を入力 → QRコード生成
4. 画像保存 → ラベルシールに印刷

#### 方法4: ラベルプリンターで直接印刷

Brother QL-800 などのラベルプリンターをお持ちの場合:

1. 付属ソフト（P-touch Editor 等）を起動
2. 「バーコード」挿入 → QRコード形式を選択
3. バーコード番号を入力
4. ラベルに直接印刷

### ラベルの貼り方

**推奨ラベル:**
- サイズ: 30mm x 30mm 以上
- 素材: 耐水性のあるラベルシール推奨
- 用紙: A4ラベルシール（100均でも購入可能）

**貼付位置:**

| 商品カテゴリ | 推奨貼付位置 |
|---|---|
| トップス | 内側の洗濯タグ付近 / 値札タグ |
| ボトムス | ウエスト内側タグ付近 / 値札タグ |
| アウター | 内ポケット付近 / 値札タグ |
| シューズ | 靴底の裏 / 箱がある場合は箱 |
| バッグ | 内ポケット / 値札タグ |
| アクセサリー | 台紙 / 値札タグ |

**ラベルの記載例:**

```
┌─────────────────────┐
│  [QRコード]  ¥4,980  │
│                      │
│  HKM-20260315-0001   │
│  Levi's 501 / M      │
└─────────────────────┘
```

### 販売時のスキャン手順

1. 販売（レジ）画面を開く
2. 「QR / バーコードで追加」ボタンをタップ
3. カメラが起動 → 商品のQRコードを枠内に合わせる
4. 自動で読み取り → 商品がカートに追加される
5. 複数商品は再度スキャンして追加
6. 全商品追加後「会計へ進む」→ 決済

**スキャンがうまくいかない場合:**
- QRコードの汚れ・破損を確認
- カメラとの距離を10〜20cmに調整
- 照明を確認（暗い場所では読み取りにくい）
- 手動検索に切り替え、バーコード番号を直接入力

### よくある質問

**Q: QRコードとバーコード、どちらを使うべき？**
A: QRコードを推奨。小さく印刷しても読み取り精度が高いです。

**Q: QRコードの中身は何を入れる？**
A: システムが発行するバーコード番号（例: `HKM-20260315-0001`）をそのまま入力。URLにする必要はありません。

**Q: QRコードが剥がれた場合は？**
A: 在庫管理画面でバーコード番号を確認し、新しいQRコードを作成して貼り直してください。

**Q: 対応ブラウザは？**
A: QRコード読み取りは **Google Chrome** / **Microsoft Edge** 最新版で動作します。
